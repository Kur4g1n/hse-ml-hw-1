# Car Price Prediction

ML-модель регрессии для предсказания цен на подержанные автомобили, выполненная в рамках домашнего задания по курсу ML в ВШЭ.

## Установка
```bash
pip install -r requirements.txt
```

## Использование

Запуск Streamlit-приложения:
```bash
streamlit run app.py
```
